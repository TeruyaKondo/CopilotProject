基本的な戻り値

関数は、計算結果や処理結果を呼び出し元に返すために戻り値を使用します。以下は基本的な戻り値の例です：

int add(int a, int b) {
    return a + b; // aとbの合計を返す
}

int main() {
    int result = add(5, 3);
    std::cout << "Result: " << result << std::endl; // Result: 8
    return 0;
}

高度な戻り値

関数は、オブジェクトやポインタを戻り値として返すこともできます。以下はオブジェクトを戻り値として返す例です：

class Complex {
public:
    double real, imag;

    Complex(double r, double i) : real(r), imag(i) {}

    Complex add(const Complex& other) {
        return Complex(real + other.real, imag + other.imag); // 新しいComplexオブジェクトを返す
    }
};

int main() {
    Complex c1(1.0, 2.0);
    Complex c2(2.5, 3.5);
    Complex result = c1.add(c2);

    std::cout << "Result: " << result.real << " + " << result.imag << "i" << std::endl; // Result: 3.5 + 5.5i
    return 0;
}
