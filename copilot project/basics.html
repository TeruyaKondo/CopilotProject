出力

C++では、標準出力ストリームを使ってコンソールにメッセージを表示します。以下は出力の基本的な例です：

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

この例では、`std::cout`を使って文字列「Hello, World!」をコンソールに出力しています。`std::endl`は改行を意味します。

変数とデータ型

C++では、変数を使ってデータを保存します。以下は基本的なデータ型の例です：

int number = 10;
float decimal = 3.14;
char letter = 'A';
std::string text = "Hello, World!";

浮動小数点数(double・float)
文字・文字列(char)
文字列(string)

制御文

制御文を使って、プログラムの流れを制御します。以下はif文の例です：

if (number > 5) {
    std::cout << "Number is greater than 5";
} else {
    std::cout << "Number is 5 or less";
}

while文

while文は、条件が真である間、繰り返し処理を行います。以下はwhile文の例です：

int count = 0;
while (count < 5) {
    std::cout << "Count: " << count << std::endl;
    count++;
}

do while文

do while文は、少なくとも一度は処理を実行し、その後条件が真である間、繰り返し処理を行います。以下はdo while文の例です：

int count = 0;
do {
    std::cout << "Count: " << count << std::endl;
    count++;
} while (count < 5);
