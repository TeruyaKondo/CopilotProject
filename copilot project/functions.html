関数の定義と呼び出し

関数を使って、コードを再利用可能にします。以下は関数の基本的な定義と呼び出しの例です：

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    std::cout << "Result: " << result;
    return 0;
}

簡単な関数定義

以下は、引数を取らず、固定のメッセージを出力する簡単な関数の例です：

void greet() {
    std::cout << "Hello, World!" << std::endl;
}

int main() {
    greet(); // Hello, World! と出力される
    return 0;
}

少し高度な関数定義

以下は、クラス内で定義されたメンバ関数の例です。メンバ関数は、クラスのオブジェクトに対して操作を行います：

class Calculator {
public:
    int add(int a, int b) {
        return a + b;
    }

    int subtract(int a, int b) {
        return a - b;
    }
};

int main() {
    Calculator calc;
    std::cout << "Addition: " << calc.add(5, 3) << std::endl; // Addition: 8
    std::cout << "Subtraction: " << calc.subtract(5, 3) << std::endl; // Subtraction: 2
    return 0;
}
