クラスとオブジェクト

クラスを使って、オブジェクトを定義します。以下はクラスの例です：

class Animal {
public:
    std::string name;
    int age;

    void speak() {
        std::cout << "Hello, I am " << name << " and I am " << age << " years old.";
    }
};

int main() {
    Animal dog;
    dog.name = "Buddy";
    dog.age = 3;
    dog.speak();
    return 0;
}
